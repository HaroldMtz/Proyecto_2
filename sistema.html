Tecnologías Utilizadas

En este curso, hemos utilizado las siguientes tecnologías clave:

React

React es una biblioteca de JavaScript para construir interfaces de usuario interactivas y reutilizables. A lo largo del curso, hemos aprovechado las capacidades de React para desarrollar componentes modulares y dinámicos, facilitando la creación de interfaces de usuario sofisticadas.

Algunas ventajas de React incluyen su eficiencia, su capacidad de renderizado virtual, y su enfoque en la creación de aplicaciones de una sola página (SPA).

GitHub

GitHub es una plataforma de desarrollo colaborativo que utiliza el control de versiones con Git. Hemos aprovechado GitHub para gestionar el código fuente de nuestro proyecto, colaborar en equipo, realizar seguimiento de problemas y realizar despliegues.

Entre las ventajas de GitHub se encuentran el seguimiento de cambios, la colaboración eficiente entre desarrolladores, y la posibilidad de integración continua con otras herramientas de desarrollo.
